NAME

fcgui − Tcl/Tk based interface to the fc chord naming program

SYNOPSIS

fcgui [ resolverprogram ]

DESCRIPTION

fcgui is a Tcl/Tk based graphical user interface to the fc program. The resolverprogram parameter is the pathname of the fc utility if it is not in the execution path.

fcgui presents a fretboard where the user marks the fret posistions on the fretboard by clicking with the mouse. The left mouse button sets the mark and the right mouse button removes it. Left of each string is the fret position indicated by a number, or ’x’ if unused. To get from open string to unused, click right mouse button on the string.
